# Основные команды проекта

## 1. Команды главного меню (MainMenu.cs)

| Команда | Назначение                 | Параметры | Возможные ошибки             |
|---------|----------------------------|-----------|------------------------------|
| 1       | Открыть каталог видеокарт  | -         | -                            |
| 2       | Управление корзиной        | -         | Требуется авторизация        |
| 3       | Просмотр истории заказов   | -         | Требуется авторизация        |
| 4       | Авторизация/регистрация    | -         | Неверные учетные данные      |
| 0       | Выход из приложения        | -         | -                            |

## 2. Команды каталога (CatalogMenu.cs)

| Команда | Назначение               | Параметры                     | Валидация                     |
|---------|--------------------------|-------------------------------|-------------------------------|
| 1       | Добавить в корзину       | ID товара, количество         | Проверка ID, доступного количества |
| 2       | Применить фильтры        | Минимальная/максимальная цена | Корректность диапазона цен    |
| 0       | Вернуться назад          | -                             | -                             |

## 3. Команды корзины (CartMenu.cs)

| Команда | Назначение         | Логика работы                          | Ошибки                  |
|---------|--------------------|----------------------------------------|-------------------------|
| 1       | Оформить заказ     | Проверка наличия товаров, создание заказа | Недостаточно товаров    |
| 2       | Удалить товар      | Удаление по ID из корзины              | Неверный ID товара      |
| 3       | Изменить количество| Обновление количества товаров          | Превышение доступного количества |
| 0       | Назад              | Возврат в главное меню                 | -                       |

## 4. Команды заказов (OrdersMenu.cs)

| Команда | Назначение       | Бизнес-логика                     | Данные     |
|---------|------------------|-----------------------------------|------------|
| 1       | Просмотр заказа  | Загрузка деталей заказа по ID     | ID заказа  |
| 2       | Повторить заказ  | Создание копии старого заказа     | ID заказа  |
| 0       | Назад            | Возврат в главное меню            | -          |

## 5. Команды авторизации (AuthMenu.cs)

| Команда | Назначение    | Поля ввода                              | Валидация               |
|---------|---------------|-----------------------------------------|-------------------------|
| 1       | Вход          | Email, пароль                           | Формат email, длина пароля |
| 2       | Регистрация   | Email, пароль, подтверждение            | Совпадение паролей      |
| 0       | Назад         | -                                       | -                       |

## 6. Сервисные команды (GpuService.cs)

| Метод          | Назначение                 | Параметры        | Исключения           |
|----------------|----------------------------|------------------|----------------------|
| GetAllGpus()   | Получить все видеокарты    | -                | -                    |
| GetGpuById()   | Найти по ID                | ID товара        | GpuNotFoundException |
| FilterGpus()   | Фильтрация по параметрам   | Цена, производитель | -                    |

## 7. Команды валидации (InputValidator.cs)

| Метод              | Назначение         | Пример использования              | Обработка ошибок       |
|--------------------|--------------------|-----------------------------------|------------------------|
| GetIntInput()      | Целочисленный ввод | `GetIntInput("Введите ID: ", 1, 100)` | Повторный запрос       |
| GetDecimalInput()  | Ввод дробных чисел | `GetDecimalInput("Цена: ", 0, 1000000)` | Проверка диапазона     |
| GetNonEmptyString()| Ввод текста        | `GetNonEmptyString("Имя: ")`      | Проверка на пустоту    |

## 8. Команды отображения (ConsoleUtilities.cs)

| Метод          | Назначение             | Пример                          | Цвет    |
|----------------|------------------------|---------------------------------|---------|
| ShowError()    | Вывод ошибки           | `ShowError("Неверный ввод")`    | Красный |
| ShowSuccess()  | Успешное действие      | `ShowSuccess("Товар добавлен")` | Зеленый |
| ClearScreen()  | Очистка консоли        | `ClearScreen()`                 | -       |

## Примеры использования команд

**Добавление товара в корзину:**
```csharp
// В CatalogMenu.cs
int gpuId = InputValidator.GetIntInput("Введите ID товара: ", 1, int.MaxValue);
int quantity = InputValidator.GetIntInput("Введите количество: ", 1, 10);
CartService.AddToCart(gpuId, quantity);
